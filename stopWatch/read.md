⏱️ StopWatch
A simple stopwatch web app built with HTML, CSS, and JavaScript.

🚀 Features
Start the stopwatch

Stop/pause the stopwatch

Reset the stopwatch to 00:00:00

Easy-to-use interface


🛠️ Tech Stack
HTML for structure

CSS for styling

JavaScript for stopwatch functionality
